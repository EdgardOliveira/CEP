# CEP
 Este é um App de exemplo de como consumir dados da API ViaCEP para consulta de endereços fornecendo CEP

# Ambiente de Desenvolvimento 
 O programa foi desenvolvido utilizando:
	- Android Studio 3.5
	- Retrofit 2.7

# Referencias
	ViaCEP
	https://viacep.com.br/
	
	Retrofit
	https://square.github.io/retrofit/
